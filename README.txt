Πώς να εκτελέσετε την εφαρμογή:

1. Μεταβείτε στον κατάλογο που περιέχει τα αρχεία εφαρμογής χρησιμοποιώντας τη γραμμή εντολών.
διαδρομή cd path/to/the/application

2. Εκτελέστε την εφαρμογή εκτελώντας την ακόλουθη εντολή:
python app.py

3. Μόλις εκτελεστεί η εφαρμογή, αποκτήστε πρόσβαση σε αυτήν ανοίγοντας ένα πρόγραμμα περιήγησης ιστού και πλοηγηθείτε στο http://127.0.0.1:5000.
